# ft2 模块版本历史

## factor（因子挖掘与回测）

### v2（当前主力）

**位置**: `v2/`

**状态**: 活跃开发中

2026-05-22 重构，自包含独立包，核心模块：

| 模块 | 职责 |
|------|------|
| `pipeline.py` | 因子→回测桥接（FactorPipeline + BacktestResult） |
| `scheduler.py` | 调仓日调度（FixedScheduler / IntervalScheduler） |
| `allocator.py` | 权重分配（TopNEqualWeight / ScoreProportional / RiskParity） |
| `validator.py` | 因子检验（IC/IR/衰减/分组收益） |
| `combiner.py` | 因子组合（等权 / ExpandingIC） |
| `grid_search.py` | 网格搜索（参数 × 频率 × TopN） |
| `bo_search.py` | 贝叶斯优化搜索 |
| `gp_primitives.py` | GP 符号回归原语集 |
| `gp_evaluator.py` | GP 多频率适应度评估器 |
| `cache_store.py` | 因子值持久化缓存 |

**设计原则**：自包含 `vN/` 版本目录，禁止跨版本依赖。

---

### v1（已归档）

**位置**: `v1/`（2026-05-25 迁移）

**状态**: 只读存档，不再发展

v1 因子挖掘框架的初始版本，为因子挖掘提供完整基础设施：

| 模块 | 职责 |
|------|------|
| `base.py` | 因子基类（ABC + FactorRegistry + @factor 装饰器 + FactorMeta） |
| `calculator.py` | 批量计算引擎（依赖图 + 并行计算） |
| `validator.py` | 统计检验（IC/IR/换手率/衰减 + 分组检验） |
| `combiner.py` | 多因子组合（IC/IR 加权 + PCA 正交化） |
| `manager.py` | 因子库管理（版本控制 + YAML/pickle 持久化） |

**迁移原因**：v2 重构了 Pipeline 回测桥接层，新增调度器、分配器、搜索优化，v1 的 Factor/Validator/Combiner/Manager 不再发展。

**迁移日期**: 2026-05-25

**迁移操作**：
```
factor/base.py       → factor/v1/base.py
factor/calculator.py → factor/v1/calculator.py
factor/combiner.py   → factor/v1/combiner.py
factor/manager.py    → factor/v1/manager.py
factor/validator.py  → factor/v1/validator.py
```

**兼容性**：2026-05-25 进一步精简，`factor/__init__.py` 改为纯 docstring（零导入），所有使用者通过 `from factor.v1` 或 `from factor.v2` 显式导入。

**位置**: `v1/`（2026-05-25 迁移）

|------|------|

| `combiner.py` | 信号融合器（投票/打分/加权/自适应/时序） |
| `threshold.py` | 阈值策略（简单/分位数/Z-Score/双阈值） |
| `registry.py` | 信号注册器 + 模板管理 |
| `backtest.py` | 轻量回测 |
| `ic_analyzer.py` | IC 分析器 |
| `indicators.py` | 指标计算函数库 |
| `timeframe.py` | 多周期数据处理 |

**迁移日期**: 2026-05-25

**迁移操作**：
